## Dependencies

This script depends on [cowsay](https://en.wikipedia.org/wiki/Cowsay)
and [fortune](https://en.wikipedia.org/wiki/Fortune_(Unix)).
In case you don't have them installed already:

On Gentoo: `emerge cowsay fortune-mod-all`

On Ubuntu/Debian: `sudo apt-get install fortune cowsay`

One CentOS/Fedora: `sudo yum install fortune cowsay`
